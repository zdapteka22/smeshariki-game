# Смешарики — Битва и Догонялки

Браузерная мини-игра. Главная страница сразу запускает игру.

## Играть (рабочие ссылки)

1. **Рекомендуется (jsDelivr):** https://cdn.jsdelivr.net/gh/zdapteka22/smeshariki-game@main/index.html
2. **Игра:** https://cdn.jsdelivr.net/gh/zdapteka22/smeshariki-game@main/game.html
3. **GitHub Pages** (если включён): https://zdapteka22.github.io/smeshariki-game/
4. **Запас:** https://litter.catbox.moe/5p6bzx.html

## Если github.io показывает 404

Открой репозиторий → **Settings → Pages → Build and deployment**:
- Source: **GitHub Actions**

После этого workflow `Deploy GitHub Pages` опубликует сайт.

## Управление

| Клавиша | Действие |
|--------|----------|
| 1 / 2 / 3 | Битва / Догонялки / Голя |
| WASD | Ходьба |
| Пробел | Удар |
| Shift | Бег |
| M | Музыка |
| Q | Выйти |
| Esc | Пауза |
