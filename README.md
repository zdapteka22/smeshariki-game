## Управление

| Клавиша | Действие |
|--------|----------|
| 1 / 2 / 3 | Битва / Догонялки / Голя |
| WASD | Ходьба |
| Пробел | Удар |
| Shift | Бег |
| M | Музыка |
| Q | Выйти |
| Esc | Пауза |
